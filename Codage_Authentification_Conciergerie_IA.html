FAQ

Questions fréquentes

Comment fonctionne l'essai gratuit ?
30 jours d'accès complet à toutes les fonctionnalités. Aucun paiement automatique — votre plan passe en gratuit si vous ne souscrivez pas. Badge ⭐ PRIORITAIRE + ✓ Vérifié pendant l'essai.
Comment fonctionne la commission ?
MyHostKit prélève 15% (Gratuit), 10% (Pro) ou 5% (Business) sur chaque prestation de ménage via Stripe. Le paiement est vérifié avant toute confirmation (action verify_payment). Sans Stripe Connect : 100% arrive sur le compte MyHostKit, reversement manuel. Avec Stripe Connect : split automatique.
Comment synchroniser Airbnb/Booking ?
Dans l'app → Logement → collez le lien iCalendar. Airbnb : Annonce → Disponibilité → "Exporter le calendrier". Booking : admin.booking.com → Chambre → Calendrier → "Exporter". Un bouton d'aide avec instructions pas-à-pas est intégré dans l'app.
Si mon paiement est refusé ?
Votre plan est temporairement gelé — vous passez en FREE mais vos données sont conservées (logements, historique, avis). Bannière rouge "⚠️ Abonnement suspendu". Dès que le paiement est régularisé → plan réactivé.
La sécurité des paiements ?
Tous les paiements passent par Stripe (navigateur in-app). Le statut ne change que si Stripe confirme (verify_payment). Pas de mise à jour avant paiement réel — aucune fraude possible. Edge Function manage-subscription déployée sur Supabase.
Qu'est-ce que la boutique produits ?
25 produits dans 6 catégories (Nettoyage, Accessoires, Consommables, Hygiène, Linge, Accueil) + commande personnalisée libre (nom + prix + quantité). Commission 15% prélevée automatiquement. Paiement vérifié par Stripe avant confirmation.
